Modelo de Procesos

Estructura y flujo del sistema de gestión presupuestaria

1
Gestionar Datos Maestros
Establece la información base para el control presupuestario. Asegura que cada contrato cuente con parámetros claros y vigentes.
  • Ejecutar para cada contrato activo
  • Definir presupuesto máximo autorizado
  • Establecer plazo de vigencia
  • Incorporar correos de responsables
2
Controlar Presupuesto
Monitorea preventivamente los compromisos potenciales antes de aprobación. Rol de alerta — no impacta el presupuesto.
  • Identificar pedidos no aprobados
  • Seleccionar por estado, CECO y fecha
  • Comparar monto vs. presupuesto disponible
  • Detectar excesos potenciales
  • Comunicar excesos a responsables
3
Agotar Presupuesto
Formaliza el impacto presupuestario una vez aprobados los pedidos. Refleja el consumo efectivo de recursos.
  • Identificar pedidos aprobados
  • Seleccionar con criterios consistentes
  • Descontar monto del presupuesto
  • Actualizar presupuesto disponible real
Visión integrada del modelo
Configuración de contratos
Establece las bases financieras: presupuesto, plazo y responsables de alerta.
Control de compromisos
Monitorea gastos pendientes para anticipar riesgos de sobregiro.
Ejecución presupuestaria
Implementa el gasto planificado al descontar pedidos aprobados.
